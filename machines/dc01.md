# dc01

## Overview

Windows Server 2022 VM serving as the primary domain controller for the bugbolab.local Active Directory domain. Hosts AD DS, DNS, and Group Policy for the lab environment. Monitored via Elastic Agent shipping Windows Security Event logs to Elasticsearch.

---

## VM Specs

| Setting | Value |
|---|---|
| **VM ID** | 103 |
| **OS** | Windows Server 2022 Standard Evaluation |
| **CPU** | 2 cores (host type) |
| **RAM** | 4096MB (4GB) |
| **Disk** | 50GB (vm-storage NVMe) |
| **IP** | 192.168.1.54 (static) |
| **Hostname** | DC01 |

---

## Services Running

| Service | Port | Notes |
|---|---|---|
| Active Directory DS | 389 (LDAP), 636 (LDAPS) | Domain authentication |
| DNS | 53 | Internal DNS for bugbolab.local |
| Kerberos | 88 | Authentication protocol |
| RDP | 3389 | Remote management |
| Elastic Agent | — | Ships logs to Fleet Server |

---

## Network Configuration

Static IP configured via Windows IPv4 adapter settings:

| Field | Value |
|---|---|
| IP Address | 192.168.1.54 |
| Subnet Mask | 255.255.255.0 |
| Default Gateway | 192.168.1.1 |
| Preferred DNS | 127.0.0.1 (self) |
| Alternate DNS | 8.8.8.8 |

> DNS points to 127.0.0.1 (itself) because the DC runs its own DNS server for the bugbolab.local domain.

## Remote Access

Managed via RDP from laptop on home network:
```
mstsc → 192.168.1.54
Username: Administrator or bugbolab\Administrator
```

When remote, connect VPN first then RDP to 192.168.1.54.
